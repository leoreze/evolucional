# Evolucional - duas versões no Render

## Rotas
- `/versao1`
- `/versao2`
- `/apresentacao`

## Rodar local
```bash
npm install
npm start
```

Abrirá em:
- http://localhost:10000/versao1
- http://localhost:10000/versao2
- http://localhost:10000/apresentacao

## Deploy no Render
- Runtime: Node
- Build Command: `npm install`
- Start Command: `npm start`

## Estrutura
- `server.js`
- `package.json`
- `render.yaml`
- `.gitignore`
- `versao1/`
- `versao2/`
